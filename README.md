# Library Management System

Sistema de gestión de biblioteca desarrollado en Java.

## Funcionalidades
- Registro de libros
- Membresías
- Préstamos y devoluciones
- Gestión de saldo

## Tecnologías
- Java
- Programación Orientada a Objetos
- Interfaz gráfica (Swing)

## Descripción
Este proyecto simula el funcionamiento de una librería real, integrando lógica de negocio y estructuras de datos.
